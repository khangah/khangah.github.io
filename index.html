CarLer.co.uk (2022-2025)

CarLer is a digital platform connecting driving instructors with learners seeking driving lessons. The ecosystem included dedicated applications for instructors, learners, and admins, supporting features such as road tracking, lesson scheduling, and progress monitoring.

Key Contributions

  • Developed and maintained three applications: an Instructor App, a Learner App, and an Admin Web App for managing the platform.
  • Core features including lesson scheduling, calendar management, payment system, and GPS-based road tracking to enhance the learning experience.
  • Implemented backend services to support authentication, user management, bookings, tracking, and administrative functions.
  • Maintained the production server and PostgreSQL database, ensuring reliable performance, scalability, and security.

Technical

  • Developed cross-platform mobile applications with Flutter, using GetX for state management and Dio for HTTP requests.
  • Built backend services with Python using Django, implementing RESTful APIs for scheduling, tracking, booking features and payment via Stripe
  • Used Pydantic for robust JSON serialization and deserialization, ensuring data integrity.
  • Used Swagger for API documentation, testing, and simplifying development and integration.
  • Managed relational data using PostgreSQL, implementing schemas for instructors, learners, lessons, and tracking data.
  • Configured, deployed and maintained the production environment on a Linux server, with Nginx as a reverse proxy for scalability and security.
